Healthcare On The Go 2 is a React Native application designed to support Home Hospital fieldworkers in their daily tasks. The app provides a modern mobile workflow with essential features such as:

1. Workday time schedule and task management
2. Patient card with detailed information
3. Emergency notifications for quick response
4. Medication reporting and tracking of supplies used
5. Camera-based barcode simulation for medicine scanning 
6. Speech-to-Text integration in Finnish for fast note-taking
7. AI-generated summary simulation inside the patient card description
***********************************************************************************************
Running the Healthcare On The Go 2 app 

1. Make sure you have Node.js and npm installed. After downloading the project, open the folder and install the required packages:
"npm install"

3. Install Android Studio for the Android SDK and device tools.
   You will need to insall Android SDK Platform, Platform Tools and Build Tools.
   Add Android tools to your PATH (sdk/platform-tools)
   
4. Connect a Device
Connect via USB or WiFi using QR code in Android Studio

6. Start the App
"npx expo run:android"
  This compiles a full native build and installs it on the device.
